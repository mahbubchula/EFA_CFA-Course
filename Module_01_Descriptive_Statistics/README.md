# Module 01: Descriptive Statistics & Data Preparation

## Overview

Before any advanced analysis, data must be thoroughly examined and prepared. This module covers the essential first steps: understanding your data's distribution, checking assumptions, identifying problems, and producing publication-quality descriptive summaries.

**This module is foundational — never skip it.**

## Learning Objectives

- Compute and interpret measures of central tendency and dispersion
- Assess normality using statistical tests and visual diagnostics
- Identify and handle missing values and outliers
- Assess scale reliability using Cronbach's alpha
- Produce well-formatted descriptive statistics tables for publication
- Implement all steps in SPSS and R

## Key Techniques Covered

- Frequency distributions and histograms
- Mean, median, mode, standard deviation, variance, range
- Skewness and kurtosis (normality indicators)
- Kolmogorov-Smirnov and Shapiro-Wilk normality tests
- Q-Q plots and box plots
- Missing value analysis (MCAR, MAR, MNAR)
- Mahalanobis distance for multivariate outliers
- Cronbach's alpha and inter-item correlation
- Correlation matrix (Pearson, Spearman)
