# Module 05: Regression Analysis

## Overview

Regression analysis is fundamental to transportation engineering and behavioral research. This module covers the full spectrum of regression techniques appropriate for different outcome types and research contexts — from simple linear regression to ordinal logistic regression.

## Learning Objectives

- Apply and interpret simple and multiple linear regression
- Conduct binary and multinomial logistic regression for categorical outcomes
- Apply ordinal regression for Likert-type or ranked outcome variables
- Check and address regression assumptions
- Interpret regression coefficients, odds ratios, and effect sizes
- Implement all methods in SPSS and R

## Regression Types Covered

| Type | Outcome Variable | Example Application |
|------|-----------------|-------------------|
| Simple Linear | Continuous | Travel time → satisfaction score |
| Multiple Linear | Continuous | Multiple predictors → travel distance |
| Binary Logistic | 0/1 Binary | Mode choice: car vs. transit |
| Multinomial Logistic | 3+ categories | Mode choice: car/motorcycle/transit/walk |
| Ordinal Logistic | Ordered categories | Crash severity: minor/moderate/severe/fatal |
| Poisson Regression | Count data | Number of crashes per road segment |
| Negative Binomial | Overdispersed count | Crash frequency |
