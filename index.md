## Sample Dataset — BUTS
{: .no_toc }

All modules use the **Bangkok Urban Transportation Survey (BUTS)**, a realistic synthetic dataset designed for this course.

| Property | Detail |
|----------|--------|
| **Respondents** | 450 |
| **Constructs** | Perceived Safety · Service Quality · Environmental Attitude · Behavioral Intention · Actual Behavior |
| **Sociodemographics** | Gender, Age, Income, Mode choice, Trip characteristics |
| **Scale items** | 19 Likert items (1–5) |
| **Missing data** | Realistic MCAR pattern for practising imputation |

[📥 Download BUTS_main.csv](https://raw.githubusercontent.com/mahbubchula/EFA_CFA-Course/main/Datasets/BUTS_main.csv){: .btn .btn-primary .mr-2 }
[📖 View Codebook]({{ site.baseurl }}/Datasets/BUTS_codebook){: .btn .btn-outline }

## How to Use This Course
{: .no_toc }

{: .note }
> **Recommended progression:** `Module 00 → 01 → 02 → 03 → 04 → 05 → 06`
> Each module builds on the previous. Do not skip Module 01 (data preparation).

Each module contains:

| Resource | Description |
|----------|-------------|
| 📖 **Lecture Notes** | Complete theory with examples from transportation research |
| 🔬 **SPSS Guide** | Step-by-step with exact menu paths and output screenshots |
| 💻 **R Script** | Fully annotated, copy-paste ready code |
| 📊 **SmartPLS Guide** | PLS-SEM workflow (Modules 03–04) |
| 📝 **Exercises** | Practice problems with real-world data |
