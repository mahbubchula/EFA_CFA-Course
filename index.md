# Advanced Statistical Methods
{: .fs-9 }

### EFA · CFA · CB-SEM · Regression Analysis
{: .fs-6 .fw-300 }

A complete, hands-on course for transportation engineering, behavioral research, accident analysis, and travel survey research using **SPSS**, **R**, and **SmartPLS 4.0**.
{: .fs-5 }

[Get Started — Module 00]({{ site.baseurl }}/Module_00_Introduction/){: .btn .btn-primary .fs-5 .mb-4 .mb-md-0 .mr-2 }
[View on GitHub](https://github.com/mahbubchula/EFA_CFA-Course){: .btn .fs-5 .mb-4 .mb-md-0 }

## What You Will Learn

| Module | Topic | Software |
|--------|-------|----------|
| [00 — Introduction]({{ site.baseurl }}/Module_00_Introduction/) | Research design, measurement theory, latent variables | — |
| [01 — Descriptive Statistics]({{ site.baseurl }}/Module_01_Descriptive_Statistics/) | Central tendency, normality, reliability, missing data | SPSS · R |
| [02 — Exploratory Factor Analysis]({{ site.baseurl }}/Module_02_EFA/) | Factor extraction, rotation, parallel analysis, item reduction | SPSS · R |
| [03 — Confirmatory Factor Analysis]({{ site.baseurl }}/Module_03_CFA/) | Model fit, convergent/discriminant validity, measurement invariance | SPSS AMOS · R · SmartPLS |
| [04 — CB-SEM]({{ site.baseurl }}/Module_04_CB_SEM/) | Full structural models, mediation, moderation | SPSS AMOS · R · SmartPLS 4.0 |
| [05 — Regression Analysis]({{ site.baseurl }}/Module_05_Regression/) | Linear, logistic, ordinal, count data regression | SPSS · R |
| [06 — Case Studies]({{ site.baseurl }}/Module_06_Case_Studies/) | Mode choice, road safety, AV acceptance, crash severity | SPSS · R · SmartPLS |

## Software Requirements

| Software | Purpose | Cost |
|----------|---------|------|
| **IBM SPSS Statistics 25+** | Descriptive stats, EFA, Regression | Licensed |
