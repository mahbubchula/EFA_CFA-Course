# Statistical Glossary

> **Course: Advanced Statistical Methods for Transportation & Behavioral Research**
> *Instructor: Dr. Mahbub Hassan, Chulalongkorn University*

---

| Term | Definition | Module |
|------|-----------|--------|
| **Alpha (α) — Cronbach's** | Measure of internal consistency reliability; ranges 0–1; ≥ 0.70 acceptable | 01 |
| **AVE (Average Variance Extracted)** | Proportion of variance in indicators explained by their latent construct; ≥ 0.50 required for convergent validity | 03 |
| **Bartlett's Test** | Tests if correlation matrix is an identity matrix (all correlations = 0); p < .05 required for EFA | 02 |
| **Bootstrapping** | Resampling method for estimating standard errors and CIs; used for mediation testing | 04 |
| **CB-SEM** | Covariance-Based SEM; uses ML estimation; for theory testing with established factor structures | 04 |
| **CFA (Confirmatory Factor Analysis)** | Tests pre-specified factor structure against data; uses fit indices | 03 |
| **CFI (Comparative Fit Index)** | Model fit index; ≥ 0.90 acceptable, ≥ 0.95 good; compares to null model | 03, 04 |
| **Common Method Variance (CMV)** | Variance due to measurement method rather than constructs; potential bias in self-report surveys | 00 |
| **Communality (h²)** | Proportion of item variance explained by common factors in EFA | 02 |
| **Composite Reliability (CR)** | Reliability index based on factor loadings; preferred over Cronbach's α in CFA; ≥ 0.70 | 03 |
| **Construct** | Abstract theoretical concept operationalized through measured indicators | 00 |
| **Convergent Validity** | Evidence that items measuring the same construct correlate highly; assessed via loadings, AVE, CR | 03 |
| **Cross-loading** | When a factor analysis item loads substantially on two or more factors; ≥ 0.30 on secondary factor is problematic | 02 |
| **Discriminant Validity** | Evidence that constructs measuring different things are distinct; assessed via HTMT | 03 |
| **EFA (Exploratory Factor Analysis)** | Data-driven method to discover latent factor structure; used in scale development | 02 |
| **Effect Size** | Measure of practical significance (Cohen's d, f², R², OR); complements p-values | 05 |
| **Eigenvalue** | Variance explained by a factor; Kaiser criterion: retain factors with eigenvalue > 1 | 02 |
| **Endogenous Variable** | Variable explained by other variables in the model (has arrows pointing to it) | 00 |
| **Exogenous Variable** | Variable not explained by the model (source of arrows, no incoming paths) | 00 |
| **Factor Loading** | Correlation between an item and its latent factor; ≥ 0.50 preferred | 02, 03 |
| **Formative Measurement** | Indicators define/cause the latent construct (opposite of reflective) | 00 |
| **Fornell-Larcker Criterion** | Traditional discriminant validity test: √AVE > inter-construct correlations | 03 |
| **Goodness of Fit** | How well a model reproduces the observed covariance structure | 03, 04 |
| **HTMT (Heterotrait-Monotrait Ratio)** | Modern discriminant validity criterion; < 0.85 required | 03 |
| **Indirect Effect** | Effect of X on Y transmitted through a mediator M; = a × b | 04 |
| **Indicator** | Observed/measured item reflecting a latent construct | 00 |
| **Intercept** | Expected value of Y when all predictors are zero; β₀ in regression | 05 |
| **KMO (Kaiser-Meyer-Olkin)** | Measure of sampling adequacy for EFA; ≥ 0.60 required | 02 |
| **Kurtosis** | Shape of distribution; excess kurtosis = 0 for normal; |kurtosis| < 7 acceptable | 01 |
| **Latent Variable** | Unobserved theoretical construct inferred from observed indicators | 00 |
| **Likert Scale** | Ordered response scale (e.g., 1=Strongly Disagree to 5=Strongly Agree) | 00 |
| **Listwise Deletion** | Removing cases with any missing value from analysis | 01 |
| **MAR (Missing at Random)** | Missingness depends on observed variables; multiple imputation appropriate | 01 |
| **MCAR (Missing Completely at Random)** | Missingness has no pattern; listwise deletion unbiased | 01 |
| **Mahalanobis Distance (D²)** | Multivariate distance measure for detecting multivariate outliers | 01 |
| **Measurement Error** | Random variation in observed scores not due to the construct being measured | 00 |
| **Measurement Invariance** | Testing whether construct operates the same way across groups | 03 |
| **Mediation** | Variable M transmits the effect of X on Y; indirect effect = a × b | 04 |
| **ML (Maximum Likelihood)** | Estimation method assuming multivariate normality; default in CFA/SEM | 03, 04 |
| **MLR** | Robust ML estimator (Yuan-Bentler); corrects for non-normality | 03 |
| **Modification Index (MI)** | Expected improvement in χ² if a constrained parameter is freed | 03 |
| **Moderation** | Effect of X on Y depends on level of Z (the moderator) | 04 |
| **Multicollinearity** | High correlation among predictors; VIF > 10 is problematic | 05 |
| **Negative Binomial Regression** | Regression for overdispersed count outcomes | 05 |
| **Oblique Rotation** | Factor rotation allowing factors to correlate (e.g., Oblimin, Promax) | 02 |
| **Odds Ratio (OR)** | exp(β) in logistic regression; ratio of outcome odds for 1-unit predictor change | 05 |
| **Ordinal Regression** | Regression for ordered categorical outcomes (e.g., severity) | 05 |
| **Orthogonal Rotation** | Factor rotation assuming factors are uncorrelated (e.g., Varimax) | 02 |
| **PAF (Principal Axis Factoring)** | EFA extraction method modeling common variance; preferred for latent constructs | 02 |
| **Parallel Analysis** | Best method for determining number of factors; compares actual eigenvalues to random | 02 |
| **Path Coefficient (β)** | Standardized regression coefficient in SEM; indicates direction and strength of effect | 04 |
| **PCA (Principal Component Analysis)** | Data reduction method; NOT true factor analysis; models total variance | 02 |
| **PLS-SEM** | Partial Least Squares SEM; used in SmartPLS; prediction-oriented | 03, 04 |
| **PLSc** | Consistent PLS; corrects PLS for attenuation bias; approximates CB-SEM | 03, 04 |
| **R²** | Coefficient of determination; proportion of variance explained | 04, 05 |
| **Reflective Measurement** | Latent construct causes indicators; standard in behavioral research | 00 |
| **Reliability** | Consistency of measurement over time, items, and methods | 01 |
| **RMSEA** | Root Mean Square Error of Approximation; fit index; ≤ 0.08 acceptable, ≤ 0.06 good | 03, 04 |
| **Scale** | Multi-item instrument measuring a latent construct | 00 |
| **Scree Plot** | Plot of eigenvalues used to determine number of factors | 02 |
| **SEM (Structural Equation Modeling)** | Framework combining measurement model (CFA) and structural model (path analysis) | 04 |
| **Skewness** | Asymmetry of distribution; |skewness| < 2 acceptable for most analyses | 01 |
| **SRMR** | Standardized Root Mean Square Residual; fit index; ≤ 0.08 acceptable | 03, 04 |
| **Structural Model** | Component of SEM specifying causal paths between latent constructs | 04 |
| **t-value** | Test statistic = estimate / standard error; |t| > 1.96 significant at p < .05 | All |
| **TLI (Tucker-Lewis Index)** | Model fit index; ≥ 0.90 acceptable, ≥ 0.95 good; penalizes complexity | 03, 04 |
| **Tolerance** | 1/VIF; < 0.10 indicates severe multicollinearity | 05 |
| **Validity** | Whether we measure what we intend to measure | 00, 03 |
| **Varimax** | Orthogonal factor rotation; maximizes variance of squared loadings | 02 |
| **VIF (Variance Inflation Factor)** | Multicollinearity indicator; < 5 acceptable, < 10 tolerable, > 10 problematic | 05 |
| **χ² / df ratio** | Normed chi-square; ≤ 3.0 acceptable for model fit | 03, 04 |
