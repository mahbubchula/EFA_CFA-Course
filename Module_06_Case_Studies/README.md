# Module 06: Applied Case Studies

## Overview

This module applies all techniques from Modules 01–05 to complete, realistic research scenarios drawn from transportation engineering, behavioral research, accident analysis, and survey research. Each case study follows the complete analytical workflow from data preparation to publication-ready reporting.

## Case Studies

| # | Title | Domain | Methods Used |
|---|-------|--------|-------------|
| CS01 | [Urban Transit Mode Choice](./CS01_Transit_Mode_Choice.md) | Transportation behavior | EFA → CFA → SEM (TPB model) |
| CS02 | [Motorcycle Safety Behavior](./CS02_Motorcycle_Safety.md) | Accident research | Scale development, CFA, SEM (PMT) |
| CS03 | [Autonomous Vehicle Acceptance](./CS03_AV_Acceptance.md) | Emerging tech | TAM-based CFA + SEM |
| CS04 | [Road Crash Severity Analysis](./CS04_Crash_Severity.md) | Accident analysis | Ordinal logistic regression |
| CS05 | [Green Travel Behavior](./CS05_Green_Travel.md) | Environmental behavior | VBN-based SEM, SmartPLS |

## Learning Objectives

- Apply the complete EFA → CFA → SEM workflow on a realistic dataset
- Make appropriate modeling decisions at each stage
- Interpret results in the context of a specific research domain
- Write publication-quality results sections
- Identify and address common challenges in real data
