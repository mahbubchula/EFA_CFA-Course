# Module 04: Covariance-Based SEM (CB-SEM)

## Overview

Structural Equation Modeling (SEM) extends CFA by adding **structural paths** between latent constructs — allowing you to test causal relationships, mediation, and moderation while accounting for measurement error. CB-SEM (using AMOS or lavaan) is the gold standard for theory testing in transportation and behavioral research.

**Why CB-SEM?**
- Tests theory-based causal models with latent variables
- Accounts for measurement error in all variables simultaneously
- Tests mediation and moderation rigorously
- Provides global model fit assessment

## Learning Objectives

- Specify a full structural equation model (measurement + structural components)
- Estimate and assess the structural model
- Test direct, indirect (mediation), and total effects
- Perform mediation analysis (bootstrapped CIs)
- Test moderation (interaction) effects
- Use AMOS, R/lavaan, and SmartPLS 4.0
